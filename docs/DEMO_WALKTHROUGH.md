# Demo Walkthrough

Use this flow when presenting the project.

1. Open `index.html`.
2. Click `Run skincare demo`.
3. Explain the brand input and campaign objective.
4. Walk through the ranked creator shortlist.
5. Highlight the 7-factor scoring breakdown.
6. Open the campaign planner view.
7. Explain the synthetic data scope and production data-source plan.
8. Close with ethics, privacy, and human review.

The Python dashboard can also be shown with:

```bash
cd engine-python
streamlit run app.py
```
