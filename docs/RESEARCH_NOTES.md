# Research Notes

This project is based on common influencer marketing workflow needs:

- Teams need to move from manual creator discovery to repeatable shortlisting.
- Follower count alone is a weak proxy for creator quality.
- View-based engagement, content relevance, commerce readiness, and brand safety should be scored separately.
- Audience demographics are often incomplete, so the system should show confidence levels instead of pretending all signals are precise.
- Human review should remain part of the final decision.

Potential data-source categories:

- Public creator profile and content data
- Authorized API data
- Internal CRM and campaign history
- First-party performance feedback
- Manual research notes from strategist teams
